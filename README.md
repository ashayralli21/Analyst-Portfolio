## About

Hi, I’m Ashay! I’m a data analyst with experience working on large-scale datasets, market research, and dashboard development.

I’ve worked with data from over 100,000+ retail locations, built Tableau dashboards to track KPIs, and developed Python pipelines to analyze datasets with over 50 million records. I enjoy turning complex data into clear insights that support real business decisions.

My experience spans Excel, SQL, Python, Tableau, and SAS, and I’m especially interested in using data to improve operational performance and uncover trends.

I’m currently pursuing my Bachelor’s in Computer Science and continuing to build projects that combine technical analysis with real-world impact.
